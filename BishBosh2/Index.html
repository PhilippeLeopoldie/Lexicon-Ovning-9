BishBosh2.0

A simple demo of javascript fundamentals the 'Bish-Bosh' game serves as hands on experience demonstrating DOM manipulation and Event and input handling

Rules:

Bish-Bosh lists all numbers between 1 and a max value, with some exceptions:

If the number is evenly divisible by 'Bish number', 'Bish' is listed instead of the number.

If the number is evenly divisible by 'Bosh number', 'Bosh' is listed instead of the number.

If the number is evenly divisible by both Bish and Bosh number, 'Bish-Bosh' is written instead of the number.
